¿Qué es el DOM?

El Document Object Model (DOM) es una interfaz de programación que permite manipular documentos HTML y XML como una estructura jerárquica de nodos.

Es la representación estructurada de un documento y actúa como puente entre la página web y los lenguajes de programación como JavaScript.

Estructura del DOM

La estructura del DOM está integrada por nodos jerárquicos que tienen relaciones de padre, hijos y hermanos.

  • Documento: Nodo raíz.
  • Elementos: Etiquetas HTML como <div>.
  • Atributos: Propiedades de los elementos.
  • Texto: Contenido dentro de las etiquetas.
  • Comentario: Comentarios en el código HTML.
